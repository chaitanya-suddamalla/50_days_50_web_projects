# Image Search Engine

A lightweight image search application that uses JavaScript to query an external image API and display results in a responsive gallery layout.

## Features

- Search images by keyword
- Responsive gallery layout
- Hover and preview interactions
- Clean UI for quick visual search

## Usage

1. Open `index.html` in a browser.
2. Type a search term in the query box.
3. Submit to view image results.

## Notes

- Requires an API key if the original implementation uses an external image service.
- Works best with modern browsers.
