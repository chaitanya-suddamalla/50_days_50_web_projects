## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/chaitanya-suddamalla/50_days_50_web_projects.git
```

2. Open `index.html` in your browser or run a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .
```
